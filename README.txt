this game is pretty simple. Make sure all the original files are in the same directory when running the game.

Controls:
W or Up Arrow -> Walk up
A or Left Arrow -> Walk left
S or Down Arrow -> Walk down
D or Right Arrow -> Walk right

M -> Toggle sounds

Left Click -> Open Dialogue
Spacebar -> Advance dialogue
Escape -> Close dialogue


Note that when clicking, you must click on a person and be in a 150px range of them.
